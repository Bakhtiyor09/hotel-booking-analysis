# Hotel Booking Data Analysis

## Project Overview
This project analyzes hotel booking data to identify booking patterns, cancellations, and seasonal trends.

## Dataset
- Hotel booking dataset (CSV)
- Includes hotel type, lead time, booking status, and customer details

## Tools Used
- Excel
- SQL
- R (ggplot2, dplyr)
- GitHub

## Key Questions
- What factors affect booking cancellations?
- Which months have the highest number of bookings?
- How does lead time impact cancellations?

## Results
- City hotels have higher cancellation rates
- Longer lead times are strongly associated with cancellations
- Seasonal peaks observed in summer months
